# frontend-starter
Basic fronted starter.

## Available scripts:

1. `npm start` - start dev server on `localhost:3000`
2. `npm test` - start testing
3. `npm run build` - build sources into `/build` folder

## Main libraries:
1. `React`
2. `React Router`
3. `Redux`
4. `React Redux Router`
5. `MaterialUI@next`
6. `Redux-sagas`
7. `Redux-form`
8. `Immutable`
9. `React-Jss`
10. `I-18next`
